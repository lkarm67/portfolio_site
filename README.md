## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/lkarm67/portfolio_site.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
npm start
```
